Verwijzen voor extern consult ()
Versturen van een verzoek (zowel digitaal als papier) van een cliënt/zorgverlener om een zorgverlener binnen de (andere) instelling in te schakelen. Uitgaande verwijzing impliceert ook het daadwerkelijk overplaatsen van een cliënt en/of het organiseren van nazorg door een andere zorgaanbieder.
