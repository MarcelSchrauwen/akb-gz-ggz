Keten-/ netwerkzorg ()
Een (regionale) organisatie heeft de verantwoordelijkheid voor de zorg van een patiëntengroep, zoals CVA-, diabetes- en/of CVRM-patiënten. De zorg wordt geleverd door meerdere zorginstellingen, waarbij afspraken zijn gemaakt over de zorgverlening, logistiek, kwaliteit en financiële afhandeling. Er zijn 2 vormen van verantwoordelijkheid: - Alle zorgaanbieders hebben gezamenlijke verantwoordelijkheid, of - Eén zorgaanbieder heeft de verantwoordelijkheid voor de behandeling [zie properties voor aanvullende informatie]
