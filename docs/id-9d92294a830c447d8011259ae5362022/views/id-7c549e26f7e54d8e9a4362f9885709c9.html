01. Consult / advies ()
Zorgverlener vraagt andere zorgverlener in consult. Verzender stelt gegevens beschikbaar aan ontvanger (inzage / sturen) en de ontvanger geeft advies over de behandeling. Zorg blijft onder de verantwoording van de verzender, inclusief dossiervoering en financiële verantwoording. Voor consult / advies kan de patiënt (eenmalig) worden gezien door de ontvangende zorgverlener. Eventueel kan de patiënt worden verwezen na uitvoeren van het consult.
